# Support

## Where to Ask for Help
- Usage/setup questions: open a GitHub Discussion or Issue.
- Bugs: use the bug report issue template.
- Security concerns: follow `SECURITY.md` (private report).

## Useful Details to Include
- Home Assistant version
- Installation type (OS/Supervised/Container/Core)
- Host OS and Bluetooth setup (local adapter or proxy)
- Integration options (`has_light`, `dimmable`, `direction_supported`, `poll_interval`)
- Relevant logs (with sensitive data removed)
